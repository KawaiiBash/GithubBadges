Welcome to the definitive guide for leveling up your GitHub profile. This document tracks every unlockable honor, from community participation to legendary historical milestones.

## ⚔️ The Active Quests
These achievements are currently obtainable. Step up your contribution game to see these land on your profile sidebar.

### 🎖️ Community Achievements
| Achievement | The Objective |
| :--- | :--- |
| **Pull Shark** | Successfully merge pull requests into any repository. |
| **Galaxy Brain** | Provide the winning answer to a question in GitHub Discussions. |
| **Starstruck** | Curate a project that the community loves (16+ Stars). |
| **Pair Extraordinaire** | Share the credit—co-author a merged pull request. |
| **Quickdraw** | The speedrun: Close an issue/PR within 5 minutes of creation. |
| **YOLO** | Live dangerously: Merge a PR without a formal code review. |
| **Heart On Your Sleeve** | Spread positivity by reacting to a post with a ❤️. |
| **Open Sourcerer** | Become a nomad: Merge PRs across various public repos. |

### 💎 Prestige Tiers
Don't just earn the badge—rank it up. High-volume contributors unlock specialized borders.

* **Bronze 🥉 (x2):** The first step toward mastery.
* **Silver 🥈 (x3):** Reserved for dedicated contributors.
* **Gold 🥇 (x4):** The pinnacle of achievement for that category.

## 🏛️ The Legacy Vault
These badges are **Legacy Items**. They were awarded for specific historical events and are no longer obtainable by new users.

* **Arctic Code Vault Contributor:** Awarded to those whose code was preserved for 1,000 years in the 2020 GitHub Archive Program.
* **Mars 2020 Contributor:** Granted to developers who contributed to the open-source libraries used by NASA's Ingenuity Mars Helicopter.

## ⚙️ Configuration & Aesthetics

### **Emoji Skin Tones**
GitHub respects your identity. Achievements featuring hands (like *Quickdraw* or *Starstruck*) will automatically match the skin tone you set in your [Appearance Settings](https://github.com/settings/appearance).

### **Stealth Mode**
If you prefer a clean, badge-free profile, you can opt-out at any time:
1. Go to your [Profile Settings](https://github.com/settings/profile).
2. Scroll to the **Achievements** section.
3. Uncheck **"Show Achievements on my profile"**.
